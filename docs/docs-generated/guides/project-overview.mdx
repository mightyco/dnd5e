# D&D 2024 Combat Simulator

Welcome to the documentation and analysis portal for the **D&D 2024 Combat Simulator**. 

## Purpose
This project is a scientific engine built in Ruby to analyze class balance, party composition, and mathematical trade-offs in the 2024 edition of Dungeons & Dragons.

## Core Principles

### Math Transparency
Every roll, modifier, and critical hit calculation is logged with full metadata. This portal allows you to visualize that data and verify the engine's integrity.

### Design-as-Code
We use Architecture Decision Records (ADRs) and OpenSpecs to ensure that every mechanical implementation is documented and peer-reviewed.

## Getting Started

1.  **View Specifications**: Explore the [Class Specs](/specs) to see how 2024 mechanics like Weapon Masteries are implemented.
2.  **Live Analysis**: Use the [Live Dashboard](/specs/simulation-dashboard/dashboard) to run simulations and visualize Damage Per Round (DPR) trends.
3.  **Architecture**: Review our [Decisions](/decisions) to understand why we chose specific technical paths.
